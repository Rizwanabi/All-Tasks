HTML(Hyper Text Markup Language)

HTML Introduction
The first publicly available description of HTML was a document called "HTML Tags", first mentioned on the Internet by Tim Berners-Lee in late 1991. It describes 18 elements comprising the initial, relatively simple design of HTML. Except for the hyperlink tag. HyperText Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.
Heading Tags
HTML headings are titles or subtitles that you want to display on a webpage. HTML headings are defined with the h1 to h6 tags. h1 defines the most important heading. h6 defines the least important heading.

Formatting Tags

In HTML, a number of elements are used to format text. The formatting tags are divided into two groups: physical tags, that are used to style the text (visual appearance of the text) and logical or semantic tags that add semantic value to the text parts (e. g., inform search engines for which keywords a web page should be ranked).
  • bold
  • strong
  • italic
  • emphasized
  • mark
  • quotation
  • preformatted
  • paragraph
  • subscript
  • superscript
Bold
The b tag specifies bold text without any extra importance.
Strong
The strong tag is used to define text with strong importance.
Italic
Italic is a typographic style where letters are slanted to the right, used to emphasize or distinguish text.
Emphasized
It is used to highlight the importance or stress of a particular word or phrase, making it stand out from the surrounding text.
Mark
The mark tag is an HTML element used to highlight a portion of text, typically rendering it with a yellow background color
Quotation
The quote tag is an HTML element used to indicate a quotation from another source, typically rendering it with indentation and quotation marks
Prefromatted
The pre tag is an HTML element used to define preformatted text, preserving the exact spacing, line breaks, and formatting of the content.
Paragraph
Paragraphs are used to organize and structure written content, making it easier to read and understand.
Subscript
It is commonly used in mathematical and scientific notation, such as in chemical formulas or mathematical equations.
Superscript
It is commonly used in mathematical and scientific notation, such as in exponents or footnotes, and also in dates and abbreviations.

Table Tags

The table tag is an HTML element used to define a table, consisting of rows and columns, to organize and display data in a structured format.
  • Table Row(tr)
  • Table Data(td)
  • Table Heading(th)
Table Row
The tr tag is an HTML element used to define a table row, containing one or more table cells.
Table Data
The td tag is an HTML element used to define a table data cell, containing data that is part of a table row.
Table Heading
The th tag is an HTML element used to define a table header cell, containing header information for a table column or row.

Anchor Tag

It is used to create a hyperlink to another web page, email address, or a specific section within a page.

Anchor Tag Attributes

  • href: Specifies the link destination (URL)
  • title: Provides a tooltip or additional information about the link
  • target: Specifies whether to open the link in a new tab or window

Image Tag

The img tag is an HTML element used to embed an image file into a web page, allowing visual content to be displayed alongside text.

Image Tag Attributes

  • src: Specifies the URL of the image file
  • alt: Provides a text description of the image for accessibility purposes
  • width and height: Specify the dimensions of the image
  • border: Specifies the border width around the image (deprecated in HTML5)
  • align: Specifies the alignment of the image (deprecated in HTML5)
  • title: Provides a tooltip or additional information about the image
